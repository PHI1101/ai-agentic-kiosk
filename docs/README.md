# AI Kiosk 프로젝트 분석

## 1. 개요 (Overview)

이 프로젝트는 'AI Kiosk'로, 단순한 챗봇을 넘어 사용자의 주문 과정을 처음부터 끝까지 주도하고 관리하는 **AI 에이전트** 시스템입니다. 사용자가 앱에 진입하는 순간부터 AI 에이전트를 소개해 역할을 명확히 하고, 음성 인식과 대화형 인터페이스를 통해 주문을 처리합니다.

핵심 목표는 일반 사용자는 물론, 시각/청각 장애인 등 정보 접근에 어려움을 겪는 사용자들도 편리하게 주문할 수 있는 '배리어프리(Barrier-Free)' 키오스크를 구현하는 것입니다.

## 2. 주요 기능 (Core Features)

- **AI 에이전트 소개 (시작 화면)**
  - 앱 실행 시, AI 에이전트의 역할과 기능을 소개하는 랜딩 페이지를 통해 사용자에게 친근한 첫인상을 제공합니다.
  - 사용자는 '주문 시작하기' 버튼을 통해 능동적으로 주문 프로세스를 시작합니다. (`HomePage.tsx`)

- **실시간 AI 상태 시각화 (아바타)**
  - 주문 화면에 AI 에이전트 아바타를 배치하여, 현재 AI의 상태(대기, 듣는 중, 생각 중)를 시각적으로 보여줍니다.
  - 이를 통해 사용자는 AI가 자신의 말을 듣고 있는지, 처리 중인지 직관적으로 인지할 수 있습니다. (`AiAgentAvatar.tsx`)

- **대화형 주문 플로우**
  - **음성 인식**: `react-speech-recognition`을 통해 사용자의 음성을 텍스트로 변환합니다.
  - **주문 확인**: AI가 사용자의 메뉴 요청을 인식하면, "장바구니에 담을까요?"라고 되물어 주문 실수를 방지합니다.
  - **상태 관리**: 사용자가 "네"라고 답하면 `orderStore`의 상태를 업데이트하여 장바구니에 상품을 추가합니다.

- **시각적 주문 현황 (장바구니)**
  - 사용자가 주문한 메뉴, 수량, 가격이 담긴 장바구니(주문 현황)를 화면에 실시간으로 표시합니다.
  - 총 주문 금액을 즉시 확인할 수 있으며, '주문 확정' 및 '전체 취소'가 가능합니다. (`OrderSummary.tsx`, `orderStore.ts`)

- **백엔드 연동**
  - Django 백엔드가 스타벅스, 메가커피 등 여러 프랜차이즈의 메뉴 데이터를 관리하며, 프론트엔드에서 요청한 주문 명령을 처리합니다.

## 3. 서비스 대상 (Target Audience)

이 서비스는 키오스크를 사용하는 모든 고객을 대상으로 하지만, 특히 다음과 같은 사용자들에게 특화된 가치를 제공하는 것을 목표로 합니다.

- **일반 사용자**: 음성을 통해 빠르고 편리하게 주문하고자 하는 모든 고객.
- **장애인 사용자**:
  - **시각 장애인**: 화면을 보지 않고도 음성 안내와 음성 명령만으로 메뉴 선택부터 주문까지 완료할 수 있도록 지원합니다.
  - **청각 장애인**: 명확하고 직관적인 채팅 인터페이스를 통해 소리 없이도 원활하게 주문할 수 있도록 지원합니다.
  - **신체 장애인**: 휠체어 사용자 등을 고려한 UI/UX 설계를 통해 물리적 접근성을 높입니다.

## 4. 기술 스택 (Tech Stack)

- **Frontend**:
  - React, TypeScript
  - **React Router DOM**: 시작 화면과 주문 화면 간의 페이지 라우팅
  - **Zustand**: 채팅(`chatStore`) 및 장바구니(`orderStore`) 상태 관리
  - **Material-UI**: UI 컴포넌트 및 디자인 시스템
  - **React Speech Recognition**: 음성 인식
  - **Axios**: 백엔드 API 통신

- **Backend**:
  - Django, Python
  - Django REST framework (API 구축)

- **Deployment**:
  - Vercel (Frontend), Railway (Backend) - `vercel.json` 설정으로 유추

## 5. 사업 아이템 상세 (Detailed Business Item)

### 5.1. 서비스 개요 (Service Overview)

이 AI 키오스크 서비스는 단순한 주문 시스템을 넘어, **인공지능 에이전트가 사용자의 주문 과정을 처음부터 끝까지 주도하고 관리하는 혁신적인 솔루션**입니다. 음성 인식, 자연어 처리, 그리고 직관적인 시각적 인터페이스를 결합하여, 사용자가 메뉴 탐색부터 결제 안내까지 전 과정에서 AI의 지능적인 도움을 받을 수 있도록 설계되었습니다. 특히, 정보 접근에 어려움을 겪는 사회적 약자(시각/청각/신체 장애인)를 포함한 모든 사용자가 불편함 없이 서비스를 이용할 수 있는 **배리어프리(Barrier-Free)** 환경을 구축하는 데 중점을 둡니다.

### 5.2. 필요성 및 대상 고객 (Necessity & Target Customers)

이 서비스는 다음과 같은 필요성을 충족시키고, 특정 고객층에 특화된 가치를 제공합니다.

*   **일반 사용자:**
    *   **편의성:** 음성 명령만으로 빠르고 효율적인 주문을 원하는 고객. 복잡한 키오스크 화면 조작에 피로감을 느끼는 사용자에게 직관적인 대화형 경험을 제공합니다.
    *   **시간 절약:** 대기 시간을 줄이고 신속하게 주문을 완료하고자 하는 바쁜 현대인.
*   **장애인 사용자 (배리어프리):**
    *   **시각 장애인:** AI 에이전트의 상세한 음성 안내를 통해 메뉴를 탐색하고 음성 명령으로 주문할 수 있습니다. 화면을 보지 않고도 독립적인 주문이 가능하여 정보 격차를 해소합니다.
    *   **청각 장애인:** 명확하고 직관적인 텍스트 기반 채팅 인터페이스를 통해 AI와 소통하며 주문할 수 있습니다. 시각적 피드백과 텍스트 대화를 통해 원활한 주문 경험을 제공합니다.
    *   **신체 장애인 (휠체어 사용자 등):** 물리적인 키오스크 조작의 어려움을 해소하고, 음성 또는 간편한 터치 인터페이스를 통해 접근성을 높여 독립적인 주문을 지원합니다.
*   **사업주 (매장 운영자):**
    *   **운영 효율성 증대:** 주문 처리 자동화를 통해 인건비를 절감하고, 직원은 고객 서비스에 더 집중할 수 있습니다.
    *   **고객 만족도 향상:** 모든 고객층에게 편리하고 포괄적인 서비스를 제공함으로써 브랜드 이미지를 제고하고 고객 충성도를 높입니다.
    *   **데이터 기반 의사결정:** 주문 데이터를 축적하여 메뉴 개발, 재고 관리, 마케팅 전략 수립에 활용할 수 있습니다.

### 5.3. 주요 기능 상세 (Detailed Core Functions)

이 서비스는 다음과 같은 핵심 기능을 통해 사용자에게 최적의 주문 경험을 제공합니다.

1.  **지능형 대화 에이전트:**
    *   **자연어 이해 (NLU):** 사용자의 복잡한 음성 및 텍스트 명령을 정확하게 이해하고 의도를 파악합니다. (예: "싸이버거 하나 주세요", "가장 인기 있는 메뉴 추천해줘")
    *   **대화 관리:** 이전 대화 내용을 기억하고 문맥을 유지하여 자연스럽고 유연한 대화 흐름을 이어갑니다.
    *   **맞춤형 응답:** 사용자의 질문과 요청에 따라 메뉴 정보 제공, 추천, 주문 확인 등 적절한 응답을 생성합니다.

2.  **음성 인식 및 텍스트-음성 변환 (STT & TTS):**
    *   **음성 인식 (STT):** 사용자의 음성을 실시간으로 텍스트로 변환하여 AI 에이전트가 이해할 수 있도록 합니다. (Powered by `react-speech-recognition` 및 Web Speech API)
    *   **텍스트-음성 변환 (TTS):** AI 에이전트의 응답을 자연스러운 음성으로 변환하여 시각 장애인 사용자에게 정보를 전달하고, 일반 사용자에게는 청각적 피드백을 제공합니다.

3.  **실시간 주문 관리 시스템:**
    *   **메뉴 탐색 및 선택:** AI 에이전트의 안내에 따라 메뉴를 탐색하고 음성 또는 텍스트로 선택할 수 있습니다.
    *   **장바구니 (주문 현황):** 선택된 메뉴, 수량, 가격, 총 금액을 실시간으로 화면에 표시하며, 사용자는 언제든지 주문 내역을 확인하고 수정할 수 있습니다.
    *   **주문 확정 및 취소:** 사용자의 최종 확인을 거쳐 주문을 확정하거나, 필요시 전체 주문을 취소할 수 있습니다.
    *   **다중 프랜차이즈 지원:** 스타벅스, 메가커피, 롯데리아, 맘스터치 등 다양한 프랜차이즈의 메뉴를 통합 관리하고 주문할 수 있습니다.

4.  **직관적인 사용자 인터페이스 (UI) 및 경험 (UX):**
    *   **AI 에이전트 아바타:** AI의 현재 상태(듣는 중, 생각 중, 말하는 중)를 시각적으로 표현하는 아바타를 통해 사용자는 AI와의 상호작용 상태를 직관적으로 인지할 수 있습니다.
    *   **채팅 인터페이스:** 음성 입력이 어려운 환경이나 청각 장애인 사용자를 위해 텍스트 기반의 채팅 인터페이스를 제공합니다.
    *   **배리어프리 디자인:** 큰 글씨, 명확한 색상 대비, 직관적인 아이콘 등 접근성을 고려한 디자인 원칙을 적용합니다.

5.  **백엔드 통합:**
    *   **Django REST API:** 프론트엔드와 백엔드 간의 효율적인 데이터 통신을 위한 RESTful API를 제공합니다.
    *   **메뉴 및 주문 데이터 관리:** Django 모델을 통해 프랜차이즈, 메뉴 항목, 주문, 주문 항목 등 모든 관련 데이터를 안전하게 관리합니다.
    *   **확장성:** 새로운 프랜차이즈 및 메뉴 추가가 용이하도록 설계되어 서비스 확장에 유연합니다.
